## Usage Examples

```typescript
import { debug } from "@/lib/debug";

// Basic logging
debug.info("auth", "User logged in", { userId: "123" });
debug.error("api", "Request failed", { status: 500 });

// Performance monitoring
debug.time("Database Query");
const result = await db.query();
debug.timeEnd("Database Query");

// Structured data
debug.table(users, "Active Users");

// Grouped logging
debug.group("Authentication Flow", () => {
  debug.info("auth", "Starting login");
  debug.success("auth", "Login complete");
});
```

## Configuration

```typescript
// Enable/disable categories
debug.enableCategory("auth");
debug.disableCategory("ui");

// Enable/disable levels
debug.enableLevel("debug");
debug.disableLevel("info");

// Get current config
const config = debug.getConfig();
```

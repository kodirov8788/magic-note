## Next.js 15 Migration Issues

### API Route Handler Changes
**Problem**: Next.js 15 changed API route parameter handling
```typescript
// ❌ Old (Next.js 14)
export async function GET(
  request: NextRequest,
  { params }: { params: { id: string } }
) { ... }

// ✅ New (Next.js 15)
export async function GET(
  request: NextRequest,
  { params }: { params: Promise<{ id: string }> }
) {
  const { id } = await params;
  // ... rest of handler
}
```

### Required Fixes
1. Update all dynamic route handlers to await params
2. Update middleware if using dynamic routes
3. Test all API endpoints after changes
